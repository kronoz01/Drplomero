# Dr. Plomero — Sitio oficial

Sitio estático (HTML/CSS/JS puro, sin dependencias de pago) para el negocio de
plomería, mantenimiento y electricidad Dr. Plomero, en Cuernavaca, Morelos.

## Publicar gratis en GitHub Pages

1. Crea un repositorio nuevo en GitHub (público), por ejemplo `dr-plomero-web`.
   Puede ser en la misma cuenta `kronoz01` que ya usas, sin problema — son
   repositorios independientes.
2. Sube el archivo `index.html` de esta carpeta a la raíz del repositorio
   (botón "Add file" → "Upload files").
3. Ve a **Settings → Pages**.
4. En "Branch", selecciona `main` y la carpeta `/ (root)`, luego **Save**.
5. Espera uno o dos minutos. Tu sitio quedará en:
   `https://kronoz01.github.io/dr-plomero-web/`

## Si más adelante compran un dominio propio

Cuando decidan comprar algo como `drplomero.com`, el proceso es:

1. Comprarlo en el registrador que elijan (Namecheap, Hostinger, etc — ver
   comparación de precios que ya revisamos).
2. En GitHub, ir a **Settings → Pages → Custom domain** y escribir el dominio
   nuevo.
3. En el panel del registrador, agregar los registros DNS que GitHub indique.
4. Esperar la propagación (unas horas) y listo.
